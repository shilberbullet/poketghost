# 포켓고스트 (PoketGhost)

포켓고스트는 터미널 기반의 요괴 수집 RPG 게임입니다.

## 주요 기능

- 요괴 수집 및 육성
- 전투 시스템
- 아이템 시스템
- 스테이지 진행
- 파티 관리
- 힐링 시스템
- 보상 시스템

## 개발 환경

- Windows 10
- GCC 컴파일러
- ANSI C

## 빌드 방법

```bash
gcc src/*.c -o poketghost -I include
```

## 실행 방법

```bash
./poketghost
```

## 프로젝트 구조

```
poketghost/
├── include/          # 헤더 파일
├── src/             # 소스 파일
└── README.md        # 프로젝트 설명
```
