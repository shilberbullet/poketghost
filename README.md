# 포켓요괴 (PoketGhost)

## 게임 소개
포켓요괴는 한국의 전통 요괴들을 수집하고 육성하는 턴제 RPG 게임입니다. 플레이어는 요괴들을 모아 전투를 벌이고, 아이템을 사용하여 요괴를 회복시키며, 다양한 스테이지를 클리어해 나가는 모험을 즐길 수 있습니다.

## 주요 기능

### 1. 요괴 시스템
- 다양한 한국 전통 요괴 수집 및 육성
- 각 요괴마다 고유한 특성과 능력치
- 레벨업 시스템을 통한 요괴 성장
- 파티 시스템을 통한 최대 5마리의 요괴 관리

### 2. 전투 시스템
- 턴제 전투 방식
- 요괴별 고유 기술 사용
- 상성 시스템을 통한 전략적 전투
- 기절 상태의 시각적 표시 (빨간색으로 표시)

### 3. 아이템 시스템
- 다양한 회복 아이템 사용
- 기절한 요괴에 대한 아이템 사용 제한
  - 탕국, 막걸리: 기절한 요괴에게 사용 가능
  - 제삿밥, 식혜: 기절한 요괴에게 사용 불가
- 인벤토리 시스템을 통한 아이템 관리
- 아이템 사용 시스템
  - 기술 선택창에서 뒤로가기: 요괴 선택창으로 복귀
  - 요괴 선택창에서 뒤로가기: 아이템 보상창으로 복귀
  - 아이템 실제 사용 시: 다음 스테이지로 진행
- **아이템 제한**
  - 일반 아이템: 최대 99개 보유 가능
  - 특수 아이템 (작두, 무당방울): 최대 5개 보유 가능
  - 복숭아: 요괴 인벤토리에서 최대 5개 보유 가능
  - 고대의 서적: 요괴 인벤토리에서 최대 5개 보유 가능
  - 인벤토리 슬롯: 최대 20개
- **복숭아 아이템 기능**
  - 요괴형 아이템으로 분류
  - 전투에 실제로 참여한 요괴가 인벤토리에 복숭아를 가지고 있으면 턴 종료 시 자동 회복
  - 복숭아 1개당 최대 체력의 5%씩 회복
  - 기절한 요괴는 회복되지 않음
  - 전투에 참여하지 않은 요괴는 회복되지 않음
  - 저장/불러오기 시에도 5개 제한이 유지됨

### 4. 스테이지 시스템
- 일반 스테이지와 보스 스테이지
- 5의 배수 스테이지마다 자동 저장
- 다음 스테이지 요괴 정보 저장
- 시간대별 다른 요괴 출현

### 5. 이벤트 시스템
- **랜덤 이벤트 발생**: 일반 스테이지에서 20% 확률로 이벤트 발생
- **편지 전달 이벤트**: 여인의 편지를 다른 지역에 전달하는 퀘스트
- **이벤트 완료 보상**: 목표 지역 도달 시 전 보상 지급 (100~300전)
- **이벤트 지속성**: 저장/로드 시 이벤트 상태 유지
- **이벤트 발생 시**: 요괴 대신 이벤트만 실행되어 전투 없이 진행

### 6. 이벤트 시스템 세부 기능
- **이벤트 중복 방지**: 활성 이벤트가 있을 때는 새로운 이벤트 발생 안함
- **전략적 목표 설정**: 방문하지 않은 지역을 우선적으로 이벤트 목표로 선택
- **탐험 유도**: 미방문 지역으로 안내하여 게임 진행 촉진
- **완전한 지역 탐험**: 모든 지역 방문 시에만 기존 지역 중에서 선택

### 7. 저장 시스템
- 수동 저장 기능
- 자동 저장 기능 (5의 배수 스테이지)
- 단일 저장 파일 관리
- 이벤트 상태 저장/로드 지원

### 8. 게임 설정
- 텍스트 속도 조절
- 텍스트 표시 모드 선택
- 상성 힌트 표시 설정
- 디버그 모드

### 9. 지역 시스템
- 게임 시작 시 경상도/전라도 중 한 곳을 선택하여 모험 시작
- 각 지역은 실제 한반도 지도를 기반으로 연결되어 있음
- 10스테이지마다 방문하지 않은 인접 지역 중 한 곳으로 자동 이동
- 이미 방문한 지역은 다시 방문하지 않음(저장/불러오기 시에도 유지)
- 지역 이동 시 연결된 지역 목록이 안내됨
- 기존 '초보자의 숲' 등 하드코딩 지역명 완전 제거, 실제 지역명만 사용
- 맵 보기 기능은 아직 구현되지 않음

### 10. 로그 시스템
- 게임 시작/종료 로그
- 모든 입출력 기록
- 오류 로그
- 게임 이벤트 로그

### 11. 통계 시스템
- 게임을 처음 시작할 때 사용자 이름(ID)을 입력받아 저장합니다.
- 메인 메뉴에서 통계를 선택하면 '(사용자 이름)님의 통계' 형식으로 통계 정보를 보여줍니다.
- 주요 통계 항목:
  - 동료로 만든 요괴 수
  - 쓰러트린 요괴 수
  - 클리어한 스테이지
  - 게임 클리어 횟수

### 12. Git 설정 자동화
- 로그 업로드 및 순위 업로드 시 Git 사용자 정보 자동 설정
- "Author identity unknown" 오류 자동 해결
- 사용자가 별도로 Git 설정을 할 필요 없음
- 로그 보내기 및 순위 업로드 기능 사용 시 자동으로 Git 설정 적용

### 13. 메인 메뉴
- 메인 메뉴 구성: "6. 로그 보내기", "7. 순위를 본다", "8. 종료"
- 사용자 편의성 향상을 위한 메뉴 배치 최적화

## 게임 시스템 상세 설명

### 요괴 시스템
- **요괴 등급**
  - 일반: 기본 능력치, 1-2개의 특수 기술
  - 희귀: 높은 능력치, 2-3개의 특수 기술
  - 전설: 최고 능력치, 3-4개의 특수 기술
- **요괴 특성**
  - 각 요괴마다 고유한 특성 보유
  - 전투 중 특정 조건에서 발동
  - 능력치 상승, 상태이상 회복 등 다양한 효과
- **요괴 육성**
  - 전투 승리 시 경험치 획득 (기본 경험치: 레벨당 80)
  - 레벨업 시 HP가 초기화되지 않고 증가량만큼만 추가
  - 레벨업 시 남은 경험치 유지
  - 기술 습득 및 PP 증가
- **기술 시스템**
  - 레벨업 시 새로운 기술 습득 가능
  - 최대 4개의 기술 보유 가능
  - 새로운 기술 습득 시 기존 기술 중 하나를 잊어야 함
  - 기술 선택 시 뒤로가기로 이전 메뉴로 복귀 가능
  - 기술 습득 후 PP가 최대치로 회복됨
- **최대 동료 수**
  - 기본 5마리까지 동료 요괴 보유 가능

### 전투 시스템
- **턴제 전투**
  - 스피드에 따른 행동 순서 결정
  - 각 턴마다 1개의 행동만 가능
  - 기술 사용, 아이템 사용, 도망 선택 가능
- **기술 시스템**
  - 각 요괴마다 4가지 기술 보유
  - 기술별 고유한 위력과 효과
  - 명중률에 따른 성공/실패 판정
- **PP 시스템**
  - 기술 사용 시 PP 소모
  - PP가 0이면 해당 기술 사용 불가
  - 회복 아이템으로 PP 복구 가능
- **도망 시스템**
  - 일반 스테이지에서 50% 확률로 도망 가능
  - 보스 스테이지에서는 도망 불가
  - 도망 실패 시 적의 공격 기회 제공
- **보스 스테이지**
  - 10스테이지마다 등장하는 강력한 요괴
  - 도망 불가, 부적 사용 불가
  - 승리 시 특별한 보상 제공

### 아이템 시스템
- **아이템 등급**
  - 일반: 기본적인 효과, 높은 획득 확률
  - 희귀: 강력한 효과, 낮은 획득 확률
  - 초희귀: 최고급 효과, 매우 낮은 획득 확률
- **아이템 종류**
  - 부적: 요괴 포획에 사용, 등급별 성공률 차이
  - 회복 아이템: HP, PP 회복, 상태이상 치료
  - 강화 아이템: 능력치 상승, 특수 효과 부여
- **아이템 제한**
  - 일반 아이템: 최대 99개 보유 가능
  - 특수 아이템 (작두, 무당방울): 최대 5개 보유 가능
  - 복숭아: 요괴 인벤토리에서 최대 5개 보유 가능
  - 고대의 서적: 요괴 인벤토리에서 최대 5개 보유 가능
  - 인벤토리 슬롯: 최대 20개
- **아이템 사용 시스템**
  - 아이템 사용 시 뒤로가기 로직
    - 기술 선택창에서 뒤로가기: 요괴 선택창으로 복귀
    - 요괴 선택창에서 뒤로가기: 아이템 보상창으로 복귀
    - 아이템 실제 사용 시: 다음 스테이지로 진행
- **아이템 보상 시스템**
  - 전투 승리 시 3개의 랜덤 아이템 중 선택
  - 아이템 등급에 따른 출현 확률 조정
  - 아이템 목록 초기화 비용이 스테이지 구간에 따라 다르게 적용됩니다.
    - 1~10스테이지: 100전
    - 11~20스테이지: 200전
    - 각 구간 내에서 초기화할 때마다 2배씩 증가
    - 게임 저장/불러오기 시에도 스테이지에 맞는 비용이 유지됩니다.
- **식혜(회복 아이템) 시스템 모듈화**
  - 식혜 관련 코드는 별도 모듈(`src/sikhye_system.c`, `include/sikhye_system.h`)로 분리되어 유지보수 및 확장성이 향상되었습니다.

### 스테이지 시스템
- **스테이지 진행**
  - 1~100스테이지까지 진행 가능
  - 스테이지별 다른 요괴 출현
  - 난이도 점진적 상승
- **보상 시스템**
  - 기본 보상: 100전
  - 스테이지 보너스: 10스테이지마다 50전씩 증가
  - 랜덤 보너스: 0~49전
  - 10의 배수 스테이지: 보상 2배
  - 보스 스테이지: 특별 보상 추가
- **지역 시스템**
  - 스테이지별로 다른 지역과 지형 설정
  - 지역별 특색 있는 요괴 출현
  - 지형에 따른 전투 효과 변화

### 저장 시스템
- **자동 저장**
  - 스테이지 클리어 시 자동 저장
  - 저장 실패 시 경고 메시지 출력
- **수동 저장**
  - 메뉴에서 언제든지 저장 가능
- **저장 데이터**
  - 현재 스테이지 정보
  - 보유 금액
  - 동료 요괴 정보
  - 인벤토리 아이템
  - 게임 설정

## 실행 방법

### 1. 개발 환경 준비
1. MinGW 설치 (C 컴파일러)
   - [MinGW 다운로드 페이지](https://sourceforge.net/projects/mingw/)에서 설치
   - 설치 시 `mingw32-base`와 `mingw32-gcc-g++` 패키지 선택
   - 설치 후 시스템 환경 변수 PATH에 MinGW의 bin 폴더 경로 추가
     (예: `C:\MinGW\bin`)

2. Git 설치
   - [Git 다운로드 페이지](https://git-scm.com/downloads)에서 설치

### 2. 프로젝트 다운로드
1. 명령 프롬프트(cmd) 실행
   - Windows 키 + R을 누르고 'cmd' 입력 후 실행

2. 프로젝트 폴더 생성 및 이동
#### 원하는 위치로 이동 (예: C 드라이브)
```bash
cd C:\
```

#### GitHub 저장소 클론
```
git clone https://github.com/shilberbullet/poketghost.git
```

#### 생성한 폴더로 이동
```
cd poketghost
```

### 3. 컴파일 및 실행

#### 소스 코드 컴파일
```bash
gcc -o poketghost.exe src/*.c core/state.c -I include -I core -lm
```
#### 게임 실행
```
poketghost.exe
```

### 4. 업데이트
#### 폴더로 이동
```
cd C:\poketghost
```

#### 최신버전 가져오기
```bash
git pull origin main
```

## 코드 모듈화

### 1. 소스 코드 구조
```
src/
├── main.c             # 메인 게임 로직
├── menu.c             # 메뉴 시스템
├── game.c             # 게임 상태 관리
├── input.c            # 입력 처리
├── text.c             # 텍스트 출력
├── stage.c            # 스테이지 관리
├── yokai.c            # 요괴 시스템
├── move.c             # 기술 시스템
├── battle.c           # 전투 시스템
├── party.c            # 파티 관리
├── savefile.c         # 저장/불러오기
```

### 1. 요괴 데이터 엑셀 변환 (`정리툴/yokai_to_excel.py`)

- **요괴종류 컬럼에도 종류별 색상(일반: 흰색, 보스: 연노랑, 패러독스: 연보라, 최종보스: 연빨강) 적용**
- 모든 컬럼(이름, 타입, 기술, 요괴종류 등)에서 정렬/필터 기능 사용 가능

### 2. 요괴 데이터 비교 (`정리툴/compare_yokai.py`)

- 결과는 `요괴_비교_결과.xlsx` 파일로 저장
- **사용법:**
  ```bash
  python 정리툴/compare_yokai.py
  ```
- **결과:** 같은 폴더에 `요괴_비교_결과.xlsx` 파일이 생성됩니다.

### 3. 기술 데이터 엑셀 변환 (`정리툴/skills_to_excel.py`)

### 6. 요괴 데이터 정리 툴 (`정리툴/format_yokai_data.py`)
- 요괴 데이터의 기술 텍스트를 정리하는 프로그램
- 기술 이름에서 공백 제거 (예: "강철 꼬리치기" → "강철꼬리치기")
- 기술 목록을 세미콜론(;)으로 구분하여 정리
- 정리된 데이터를 텍스트 파일로 저장

#### 사용 방법
1. `format_yokai_data.py` 파일의 `yokai_data` 변수에 정리할 요괴 데이터를 붙여넣습니다.
2. 스크립트를 실행합니다:
   ```bash
   python 정리툴/format_yokai_data.py
   ```
3. 정리된 데이터는 '정리된_요괴_데이터.txt' 파일에 저장됩니다.

#### 데이터 형식
입력 데이터는 다음 형식을 따라야 합니다:
```
이름,타입,공격력,방어력,체력,스피드,도감설명,기술1;기술2;기술3;...
```

예시:
```
도깨비,MONSTER,200,150,180,170,밤에 나타나 사람들을 놀라게 하지만 가끔씩 도움을 주기도 한다.,불꽃치기;지옥불;맹화;가시돌진;사념의박치기;강철꼬리치기;몸통박치기;번개
```

#### 주의사항
- 입력 데이터는 반드시 8개 이상의 필드를 포함해야 합니다 (이름, 타입, 공격력, 방어력, 체력, 스피드, 도감설명, 기술들)
- 빈 줄은 자동으로 무시됩니다
- 기술 이름에 포함된 모든 공백이 제거됩니다

### 7. 기술 데이터 정렬 툴 (`정리툴/skills_sorter.py`)
- 기술 데이터를 등급별로 분류하고 가나다순 정렬하는 프로그램
- 초급/중급/고급 기술로 분류
- 각 등급 내에서 기술 이름을 가나다순으로 정렬
- 정렬된 데이터를 텍스트 파일로 저장

#### 사용 방법
1. `skills_sorter.py` 파일의 `skill_data` 변수에 정리할 기술 데이터를 붙여넣습니다.
2. 스크립트를 실행합니다:
   ```bash
   python 정리툴/skills_sorter.py
   ```
3. 정렬된 데이터는 'sorted_skills.txt' 파일에 저장됩니다.

#### 데이터 형식
기술 데이터는 다음 형식을 따라야 합니다:
```
# 기술 데이터 파일
# 형식: 이름,기술의 상성,공격력,명중률,PP,설명

# 초급 기술
불꽃치기,MONSTER,25,100,20,불꽃으로 적을 공격한다
돌던지기,HUMAN,25,95,25,돌을 던져 적을 공격한다
...

# 중급 기술
바람칼,HUMAN,35,85,20,바람으로 만든 칼날을 날린다
...

# 고급 기술
지옥불,EVIL_SPIRIT,50,70,5,지옥의 불꽃으로 적을 태운다
...
```

#### 주의사항
- 기술 데이터는 반드시 등급별 헤더(# 초급 기술, # 중급 기술, # 고급 기술)를 포함해야 합니다
- 빈 줄은 자동으로 무시됩니다
- 원본 데이터 형식이 유지됩니다

### 8. 요괴 데이터 정렬 툴 (`정리툴/yokai_sorter.py`)
- 요괴 데이터를 종류별, 등급별(일반→보스→패러독스), 이름순으로 정렬하는 프로그램

- 정렬된 데이터를 텍스트 파일로 저장

#### 사용 방법
1. `yokai_sorter.py` 파일의 `yokai_data` 변수에 정리할 요괴 데이터를 붙여넣습니다.
